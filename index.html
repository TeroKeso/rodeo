Rancher Rodeo Helm Charts

Usage

helm repo add rodeo https://rancher.github.io/rodeo

This repository contains Helm charts that are used during Rancher Rodeo webinars and for demonstration purposes.

The Helm charts in this repository are not production ready and are meant for demonstration purposes only!

Charts

bookinfo (0.2.3@1.16.2)

Istio bookinfo app

loki (2.5.4@v2.2.0)

Loki: like Prometheus, but for logs.

mssql (0.1.1@1.16.0)

Installs Microsoft SQL Server in Docker

mysql-operator (0.4.5@0.4.0)

MySQL Operator from Presslabs

nfs-server-provisioner (1.2.5@2.3.0)

nfs-server-provisioner is an out-of-tree dynamic provisioner for Kubernetes. You can use it to quickly & easily deploy shared storage that works almost anywhere.

nginx (0.2.4@1.21.0)

A plain nginx

online-boutique (0.3.4@v0.2.3)

A microservice online-boutique web shop to demo Istio

quake-kube (0.2.3@v1.0.7)

Quake 3 Arena in a browser

rancher-demo (2.1.4@1.0.6)

Rancher demo application, web UI showing scale of containers

wordpress (11.0.20@5.7.3)

Web publishing platform for building blogs and websites.
